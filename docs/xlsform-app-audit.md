- Coverage: XLSForm and app JSON both contain 30 groups/sections and 449 question-bearing rows.

- Four compiled source constraints are evaluable but inert for real app inputs because their constraint values do not match the compiled control/choice list: `Id10260`, `Id10414`, `Id10414_a`, and `Id10414_b`. They cannot produce a runtime constraint validation error through valid app choices.
